Projects I Maintain
  • hydrium - A fast, ultra-low-memory, streaming JPEG XL encoder written in portable C.
  • JXLatte - A JPEG XL decoder written in pure Java.
  • The Grimoire - A searchable Dungeons & Dragons 5th Edition spellbook. This is a fork of Seph’s (ephe’s) Grimoire with several added features and tags (and various things fixed up). As of writing this, this fork is currently the most used one I could find on the internet (which is exactly as reliable of a statement as is sounds). Given the nature of D&D, I try to update the grimoire with the various errata, but I don’t guarantee that it’s perfectly correct or fully content-inclusive. Also, for the record, I’m shocked how I’ve managed to keep it maintained this long given that I don’t really know anything about Ruby or Jekyll. So if anyone wants to help me out with pull requests and such, please feel free.
  • Orange Diff - Corporate wants you to show the difference between two pictures. Designed as a replacement for diff.pics and the like, but requires you to host the images elsewhere.
  • Multiclass Spellcasting Calculator - Calculator for the number of spell slots possessed when multiclassing, in Dungeons and Dragons, 5th edition.
  • Deep Blue Sky - A discord bot written in python. Very WIP at the moment.
  • Azur Lane Utilities - Some utilities I’ve written for the mobile game Azur Lane
  • USCF Rating Estimator - An improved US Chess Rating Estimator based on performance in a tournament.
  • mpv - Cross-platform open-source command-line media player. I’m a maintainer.
